Layers
------

.. list-table::
   :header-rows: 1
   :widths: 22 78

   * - Layer
     - Responsibility
   * - Corpus
     - ``corpus/responses/{prefix}-*.md`` + front matter
   * - Quality
     - ``check`` (prefix/sop/closings), ``lint`` engines
   * - Index
     - Tantivy BM25 under ``.canonic-index/``
   * - Publish surface
     - ``convert`` (pandoc jira) / human paste
   * - Import drafts
     - ``import-jira`` → ``corpus/imports/`` only

Free Jira platform map
----------------------

canonic uses only **native** Jira REST (Cloud Free API tokens or Server/DC
PAT) - no Marketplace apps:

.. list-table::
   :header-rows: 1
   :widths: 18 48 34

   * - Command
     - Endpoint
     - Body / notes
   * - ``jira-probe``
     - ``GET /rest/api/2/myself`` (+ serverInfo)
     - reports Cloud vs Server
   * - ``import-jira``
     - ``GET …/search`` (v2, then v3/jql fallback);
       ``GET …/issue/{key}/comment``
     - comments wiki **or** ADF
   * - ``jira-comment``
     - Server: ``POST /rest/api/2/…/comment``;
       Cloud: ``POST /rest/api/3/…/comment``
     - wiki string / minimal free ADF

# Demo - how to request software

Hello,

This is a **demo** answer. To request a software install, open a ticket with:

- Package name and version (or source URL).
- License notes if the software is not fully open source.
- Any toolchain or module preferences.

We will check whether an existing module covers the need before building a
custom install.

Regards,
Support Team

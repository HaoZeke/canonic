# Project space is not a backup

Hello,

Project space on the cluster is for active project data, not long-term backup or archival storage.

- Keep only working data on project space.
- Use the documented backup or archive workflow for data you must retain.
- Remove obsolete datasets so the shared quota stays available for the project.

If you need help choosing a storage tier, open a ticket with the project and quota details.

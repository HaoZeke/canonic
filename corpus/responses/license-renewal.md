# Software license renewal workflow

Hello,

To renew a commercial software license before expiry:

1. Confirm the SKU and seat count in the asset register
2. Submit a procurement request with cost center and manager approval
3. Attach the vendor quote PDF and the previous entitlement certificate

Renewals must be filed **30 days** before the expiry date shown in the license portal. Emergency rush renewals use the code `LICENSE_RUSH` in the subject line.

Do not install cracked or evaluation keys on production hosts.

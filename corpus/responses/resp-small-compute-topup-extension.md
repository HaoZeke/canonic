# Small grant top-up or one six-month extension

Hello,

For an **active small grant**:

- Additional SBUs during the project can be requested via the service desk when they still fit the project scope. Second-line or infra review checks fit, then first line can apply the change.
- **One** six-month extension can be handled in the small-grant process when the rules are still satisfied. If the project has already used its extension, further extension is not available on that path.

For **large grants**, extensions are handled through **NWO**, not as a direct small-grant style extension by advisors.

Please state whether you need a top-up (how many SBUs and why) or an extension (current end date and whether an extension was already used), and confirm the project still matches the original scientific scope.

Regards,
